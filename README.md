# Link warm koud

## Installing
Install npm, daarna:

```console
$ git clone https://github.com/Tijs-B/chirolink-warm-koud.git
$ cd link
$ npm install
```

## Running
Om het in ontwikkelaarsmodus uit te voeren (met rebuild bij change, en local web server):
```console
$ npm run serve
```

Om te builden in productie:
```console
$ npm run build
```
